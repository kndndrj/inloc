# Ray Tracer

| Language | Lines of Code |
| :------: | :-----------: |
|   C++    |      127      |

## Running

Use GCC to compile the file and run it:

```sh
g++ ray_tracer.cpp && ./a.out
```

This should produce a file called `out.ppm`. Use a tool like GIMP to view it.
